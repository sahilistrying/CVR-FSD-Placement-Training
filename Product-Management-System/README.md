# Product Management System

This is a simple static website for a Product Management System. It serves as a landing page that provides an overview of the product and includes interactive elements.

## Project Structure 
```
product-management-website
├── index.html        # Main HTML document for the landing page
├── css
│   └── style.css     # Styles for the landing page
├── js
│   └── script.js     # JavaScript for button interactions
├── assets
│   └── images        # Directory for images used in the website
└── README.md         # Documentation for the project
```

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd product-management-website
   ```

3. Open `index.html` in your web browser to view the landing page.

## Features

- Product title and short description displayed on the landing page.
- Interactive button that triggers a JavaScript function.
